# 🏙️ zhihuiti-city (智慧体：进化城市)

An autonomous city simulation powered by the **zhihuiti** (智慧体) evolutionary engine.

### 🧪 Evolutionary Roles (v1.1)

Now featuring advanced evolutionary theories in the local simulation:

*   **Mutualist (互助者) 🤝**: `#fdd835` Yellow. Focuses on **Parks**. Logic: *Cooperation over competition.*
*   **Darwinian (达尔文派) 🐺**: `#f44336` Red. Focuses on **Towers**. Logic: *Survival of the fittest (High-budget bidding).*
*   **Hybrid (混合体) 🧬**: `#8e24aa` Purple. Focuses on **Markets**. Logic: *Adaptation to local environment.*

### 🚀 Running the simulation

```bash
npm install
npm run dev
```

### 🧠 Features

1.  **Agent Visualizer**: Live mapping of `zhihuiti` backend agents to isometric sprites.
2.  **Economic Simulation**: Agents earn budget by working near buildings and use it to build their preferred structures.
3.  **Bidding Competition**: When multiple agents target the same tile, the highest budget wins.
4.  **Thought Engine**: Real-time thoughts reflecting goals, budget status, and role-specific theories.
